Fields

public static final PSSParameterSpec DEFAULT

The default parameter specification. It specifies the following parameters:

  • message digest: "SHA-1"
  • mask generation function (mgf): "MGF1"
  • parameters for the mgf: SHA1
  • salt length: 20
  • trailer field: -1
