public interface

DataOutput

Known Indirect Subclasses
DataOutputStream, ObjectOutput, ObjectOutputStream, RandomAccessFile
DataOutputStream Wraps an existing OutputStream and writes big-endian typed data to it. 
ObjectOutput Defines an interface for classes that allow reading serialized objects. 
ObjectOutputStream A specialized OutputStream that is able to write (serialize) Java objects as well as primitive data types (int, byte, char etc.). 
RandomAccessFile Allows reading from and writing to a file in a random-access manner. 

Defines an interface for classes that are able to write big-endian typed data to some target. Typically, this data can be read in by a class which implements DataInput. Types that can be written include byte, 16-bit short, 32-bit int, 32-bit float, 64-bit long, 64-bit double, byte strings, and MUTF-8 encoded strings.
