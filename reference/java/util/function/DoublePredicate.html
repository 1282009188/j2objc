public interface

DoublePredicate

java.util.function.DoublePredicate

Class Overview

Represents a predicate (boolean-valued function) of one double-valued argument. This is the double-consuming primitive type specialization of Predicate.

This is a functional interface whose functional method is test(double).

public abstract DoublePredicate and (DoublePredicate other)

Returns a composed predicate that represents a short-circuiting logical AND of this predicate and another. When evaluating the composed predicate, if this predicate is false, then the other predicate is not evaluated.

Any exceptions thrown during evaluation of either predicate are relayed to the caller; if evaluation of this predicate throws an exception, the other predicate will not be evaluated.
