EventListenerProxy<T extends EventListener> An abstract wrapper class for an EventListener class which associates a set of additional parameters with the listener. 
HandshakeCompletedListener The listener to be implemented to receive event notifications on completion of SSL handshake on an SSL connection. 
PropertyChangeListener A "PropertyChange" event gets fired whenever a bean changes a "bound" property. 
PropertyChangeListenerProxy A class which extends the EventListenerProxy specifically for adding a PropertyChangeListener with a "bound" property. 
SSLSessionBindingListener The interface to be implemented by any object that requires notification when data objects are bound to (or unbound from) an SSLSession. 

Class Overview

A tagging interface that all event listener interfaces must extend.
